# Backend

## For backend quick-start:

You can use the [Spring Initializr](https://start.spring.io/) to create a new project with the backend dependencies.

## For backend debugging:

Other than IntelliJ IDEA, you can use the [Postman](https://www.postman.com/downloads/).
It can be used for:
- Checking if the URL is correct
- Testing request method (GET / POST / PUT / DELETE)
- Checking if the params passed correctly
- Checking if the returned data matches your expectation
